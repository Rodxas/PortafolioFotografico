# Zuzu Hernández - Portafolio de Fotografía de Modas

Portafolio profesional para Diseñadora de Modas y Fotógrafa.

## 🌸 Descripción

Portafolio artístico con colores pasteles (verde y beige), estilo "unicornio vomitado" pero profesional. Enfocado en展示 trabajos de fotografía de modas.

## 👩‍🎨 Características

- Diseño artístico con paleta de colores pasteles
- Modo oscuro/nocturno
- Lightbox para visualizar imágenes en grande
- Copiar correo electrónico con un clic
- Responsive design (móvil y escritorio)
- Animaciones suaves

## 🛠️ Tecnologías

- HTML5
- CSS3 (Variables CSS)
- JavaScript Vanilla
- ASP.NET Core (para desarrollo local)

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/TU_USUARIO/zuzu-hernandez-portafolio.git

# Navegar al directorio
cd zuzu-hernandez-portafolio

# Restaurar dependencias
dotnet restore

# Ejecutar localmente
dotnet run
```

## 📂 Estructura

```
ZuzuPortfolio/
├── wwwroot/
│   └── prototipo.html    # Archivo principal (imágenes desde Google Drive)
├── fotosDeWeb/           # Fotos locales (MainPhoto, SecondaryPhoto)
├── Program.cs
├── appsettings.json
└── ZuzuPortfolio.csproj
```

**Nota:** Las imágenes del portafolio se cargan desde Google Drive para mantener el repositorio ligero (< 25MB).
